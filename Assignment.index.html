What are Heading?

This are subtitles that you want to display on a webpage. Html provides six level of headings from, <h1>to <h6>, with <h1> being the most important and <h6>the least important.

Paragraphs in HTML

They are created using the <p>tag. They are elements that provide structure to your conntent.

Styles

You can style HTML elements using CSS. For example this paragraph has a different background color, padding and a left border. CSS helps make the webpage look more attractive and organized.
